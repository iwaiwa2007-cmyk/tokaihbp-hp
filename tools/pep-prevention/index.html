判定ロジック・出典

Version 1.1
  • 高リスク：確実なリスク因子1つ以上、または可能性のあるリスク因子2つ以上（ESGE 2020の分類に基づく）。リスク因子は相加的で、二分判定だけでは個別リスクを完全には表しません。
  • 直腸NSAIDs：禁忌がないERCP患者で周術期投与を検討。薬剤・用量・時期は添付文書および院内プロトコルを確認してください。
  • 膵管ステント：選択された高リスク状況、意図しない膵管ガイドワイヤー挿入／造影、ダブルガイドワイヤー法、乳頭切除等で検討。無理な追加膵管挿管は避けます。
  • 積極的輸液：体液過剰リスクがない場合に個別検討。NSAIDs併用時の上乗せ効果、外来運用、患者背景を考慮します。
